Introduction

This Python Documentation will give you a short overview about some simple Python stuff.

Python has a simple syntax similar to the English language.
It has syntax that allows developers to write programs with fewer lines than some other programming languages.


Hello World

To get started, simply open your text editor and later save the file as .py.

print("Hello World")


To run the file, open CMD, navigate to your file and execute it with "python hello-world.py".


Variables

Variables are containers for storing data values.

Python has no command for declaring a variable.
A variable is created the moment you first assign a value to it.


x = 5
y = "John"

You can use both single('') or double("") quotes.


Variable names are also case-sensitive, so

A = "hello"

is different variable than

a = "hello"

Here are 5 arithmetic operator that you can weork with:

  • + ~ Addition

  • - ~ Subtraction

  • * ~ Multiplication

  • / ~ Division

  • ** ~ Exponentiation


Lists

Lists are used to store multiple items in a single variable.


Lists are declared like variables, so they can be named anything.


They are created using square brackets:

list = ["apple", "banana", "cherry"]

You can then get the list printed by typing:

print(list)


Strings

Strings in python are surrounded by either single quotation marks, or double quotation marks.

'hello' is the same as "hello".


You can display a string literal with the print() function:

print("Hello")

or

print('Hello')
